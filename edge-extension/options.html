Keyboard Shortcuts (While TTS Session Is Active)

  • Arrow Left/Right: Move by 1 segment
  • Ctrl + Arrow Left/Right: Jump by configured segments
  • Home/End: Jump to first/last segment (preview)
  • Ctrl + Home/End: Jump to first/last and resume reading
  • Space: Pause/resume
  • [ / ]: Decrease/increase speech rate by configured step
  • R: Replay current segment
  • L: Toggle loop-to-top
  • A: Toggle auto-scroll
  • Ctrl + Up/Down: Cycle sent prompts when prompt box is focused
